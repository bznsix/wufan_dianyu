经过午饭老师的统计：每天BTC的振幅在1%以上的次数都有很多次，所以我么可以以1%为间距来做网格，像打鱼一样筛选到这些利益。由于我们是电动的，所以我们叫电鱼。
再加以杠杆的力量，无法想象！
但是午饭老师也说过：因为加了杠杆（一般是10倍）所以一个月最少爆仓一次。请及时计提你的收益。
最好的战绩应该是午饭老师的一个月30倍数，这个我算了，算上复利一天要13%左右收益，这个我应该无法做到。

主要参数说明：
在martin_long.py文件中：
```
params = (
    ('margin', 11), #杠杆倍数
    ('gap_time', 2*5), #分成多少份
    ('kline_interval',4*60*(60*1000)), #多少分钟
    ('long_win_ratio',0.01), # 止盈比例
    ('start_price',99999) # 启动价格
)
```
其余的参数都很好理解，解释下kline_interval
这个参数的主要作用怕是你遇到了连续下跌疯狂抄底直接给爆了，每次抄底完后会间隔你设置的这个时间再抄。（虽然该爆还是要爆）

启动的时候还是设置双向持仓，在broker中设置APIKEY,选用逐仓模式（这样不会一次给你爆完），然后启动！
